Kako bi pokrenuli nasu aplikaciju potrebno je da imate instalirano python radno okruzenje

****************************************************************************************************

Uputstvo za koristenje
1. Potrebno je u postavkama (settings) precizirati putanju do foldera sa slikama
2. Potrebno je u postavkama (settings) precizirati putanju do .txt filea koji sadrzi key words,
a moguce ih je i rucno upisati iz GUI-a
3. Svakim pritiskom na button "Recognize image" pokrenuti ce se python skripta i korisniku se vraca
odgovor generisan putem AI-a, koji ce pokrenuti aktuator preko COM porta
4. Sve slike moraju imati naziv "slika"+index


****************************************************************************************************

Tri primjera generickih slika nalaze se u folderima na putanji .../xs5/baza/

****************************************************************************************************

NAPOMENA: Kao odgovor od AI-a mozete dobiti "NA" sto znaci da ono sto se nalazi na slici ne odgovara
ponudenim key words
